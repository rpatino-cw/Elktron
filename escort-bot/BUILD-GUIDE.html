Build Guide

Every part. Where it goes. What code it runs. Follow in order, top to bottom.

Parts: 11 components Code: 6 files Build time: ~3 hours Owner: Romeo Patino / Alex Murillo
01 — Parts + Code Map
01
Raspberry Pi 5
The brain. Runs everything.
Install
  • Flash SD card with Pi OS Lite 64-bit (Bookworm)
  • Enable SSH + WiFi in Imager settings
  • Hostname: escort-bot
  • Boot, SSH in: ssh pi@escort-bot.local
Code
main.py pan_tilt.py install.sh
02
Active Cooler
Keeps Pi from thermal throttling under CV load.
Install
  • Power off Pi
  • Peel thermal tape, press heatsink onto SoC chip
  • Plug 4-pin fan header into Pi 5 fan connector
  • PWM auto-adjusts speed by temp
Code

No code. Hardware-level cooling.

03
Arducam Camera Module 3 Wide
The eyes. 120° FOV, autofocus, person detection input.
Install
  • Connect 15-to-22pin FFC cable (included)
  • Pi 5 CSI port is between the two HDMI ports
  • Lift latch, slide cable in (contacts facing board), close latch
  • Route cable down through PVC mast before connecting
Code
main.py → init_camera() main.py → detect_person()

Uses picamera2 → libcamera stack. Feeds 640x480 RGB frames to TFLite.

Test

libcamera-hello --timeout 5000

04
Arducam Pan Tilt Platform
The neck. 2 servos sweep camera for rack scanning.
Install
  • Screw camera module onto pan-tilt bracket
  • Mount platform base on top of PVC mast (zip-tie)
  • Route servo wires down inside the pipe
Pins
GPIO 12 → Pan GPIO 13 → Tilt 5V (Pin 4) → VCC GND (Pin 9)
Code
pan_tilt.py → PanTilt class pan_tilt.py → scan_rack()

AngularServo via gpiozero. Sweeps -60° to +60° tilt for full 42U rack scan.

Test

python3 pan_tilt.py --scan

05
LK-COKOINO 4WD Chassis
The legs. 4 DC motors, tank steering.
Install
  • Mount 4 motors to bottom plate with brackets
  • Shafts face outward. Attach wheels.
  • Wire LEFT pair (FL+RL) in parallel
  • Wire RIGHT pair (FR+RR) in parallel
  • Quick test: touch battery to wires → verify forward spin
Code
main.py → Robot(left=(17,27), right=(22,23)) main.py → compute_steering() main.py → drive()
06
L298N Motor Driver
The muscle. Converts Pi signals into motor power.
Install
  • Mount on top chassis plate
  • LEFT motors → OUT1 + OUT2
  • RIGHT motors → OUT3 + OUT4
  • Battery (+) → +12V terminal
  • Battery (-) → GND terminal
  • REMOVE the 5V jumper
Pins
GPIO 17 → IN1 (left fwd) GPIO 27 → IN2 (left bwd) GPIO 22 → IN3 (right fwd) GPIO 23 → IN4 (right bwd) GND → Pi GND (Pin 6) CRITICAL
Code
main.py:34 → LEFT_MOTOR = (17, 27) main.py:35 → RIGHT_MOTOR = (22, 23)
Test

python3 -c "from gpiozero import Robot; r=Robot(left=(17,27),right=(22,23)); r.forward(0.3); import time; time.sleep(1); r.stop()"

07
HC-SR04 Ultrasonic Sensor
Proximity. Emergency stop if obstacle < 30cm.
Install
  • Mount on front of chassis, facing forward
  • ECHO pin is 5V — MUST use voltage divider
  • ECHO → 1kΩ → GPIO 24 junction → 2kΩ → GND
Pins
GPIO 25 → TRIG GPIO 24 → ECHO (via divider) 5V (Pin 2) → VCC GND (Pin 6)
Code
main.py:40 → ULTRASONIC_ECHO = 24 main.py:41 → ULTRASONIC_TRIG = 25 main.py:254 → sonar.distance < STOP_DISTANCE
Test

python3 -c "from gpiozero import DistanceSensor; s=DistanceSensor(echo=24,trigger=25); print(f'{s.distance*100:.1f}cm')"

08
PVC Mast + Tee
The spine. Raises camera to mid-rack height (~4ft).
Install
  • Cut 1" pipe to ~48 inches
  • Cut two 4" stubs for tee horizontal arms
  • Glue with PVC cement (or friction-fit + tape)
  • Mount tee flat on REAR of chassis
  • Zip-tie tee arms down through adhesive mounts
  • Drill hole at pipe bottom for wire exit
Code

No code. Structural only.

09
18650 Batteries (x2)
Motor power. 7.4V through L298N.
Install
  • Charge fully before assembly
  • Insert into 2-cell holder (series = 7.4V)
  • Connect holder to L298N +12V / GND terminals
Code

No code. Powers motors only.

10
USB-C Power Bank
Pi power. Clean 5V/3A, separate from motor circuit.
Install
  • Velcro or strap to chassis (weight = stability)
  • Short USB-C cable to Pi 5
  • Do NOT power Pi from L298N 5V regulator
Code

No code. Pi power source.

11
Zip Ties + Adhesive Mounts
The skeleton. Holds everything together.
Install
  • Clean surfaces with alcohol before sticking mounts
  • 4+ mounts on chassis for mast tee arms
  • Cable management: group wires, tie along mast
  • Screw-down bases for permanent anchor points
Code

No code. Physical structure.

02 — Wiring
FULL GPIO MAP Pin 2 (5V) ─── HC-SR04 VCC Pin 4 (5V) ─── Servo VCC (both) Pin 6 (GND) ─── HC-SR04 GND + L298N GND ← COMMON GROUND Pin 9 (GND) ─── Servo GND (both) Pin 11 (GPIO 17) ─── L298N IN1 (left fwd) Pin 13 (GPIO 27) ─── L298N IN2 (left bwd) Pin 15 (GPIO 22) ─── L298N IN3 (right fwd) Pin 16 (GPIO 23) ─── L298N IN4 (right bwd) Pin 18 (GPIO 24) ─── HC-SR04 ECHO via 1kΩ+2kΩ voltage divider Pin 22 (GPIO 25) ─── HC-SR04 TRIG Pin 32 (GPIO 12) ─── Pan servo signal Pin 33 (GPIO 13) ─── Tilt servo signal
POWER LAYOUT USB-C Power Bank (5V/3A) ──── Pi 5 USB-C └── Pi 5V rail ─── HC-SR04 + Servos 18650 x2 (7.4V) ──── L298N +12V / GND └── OUT1-4 ─── 4 DC motors Two separate power sources. Connected ONLY by common GND.
03 — Flash Pi 5 SD Card
Requirements
# You need: - Raspberry Pi 5 (4GB or 8GB) - MicroSD card 32GB+ (Class 10 / A2) - SD card reader (USB or built-in) - WiFi name + password
Samsung EVO Select or SanDisk Extreme recommended. Do NOT use Bullseye, Ubuntu, or 32-bit.
Step 1 — Install Raspberry Pi Imager
brew install --cask raspberry-pi-imager
Or download: imager_latest.dmg
Step 2 — Flash settings
# In Raspberry Pi Imager: Device: Raspberry Pi 5 OS: Raspberry Pi OS (other) > Pi OS Lite (64-bit) Debian Bookworm, no desktop Storage: Your MicroSD card
Lite = headless. No desktop, all RAM for CV + motors. 64-bit required for TFLite performance.
Step 3 — OS Customisation (CRITICAL)
# When prompted "Apply OS customisation?" # Click EDIT SETTINGS: Hostname: escort-bot Username: pi Password: [your choice] WiFi SSID: [your network] WiFi Pass: [your password] Country: US Timezone: America/Chicago SSH: Enable (password auth)
Do NOT skip this. Without WiFi + SSH pre-configured, you can't connect headless. Click Save > Yes > Yes. Wait ~5-10 min.
Step 4 — First boot + SSH
# Eject SD, insert into Pi 5, plug USB-C power # Wait 60-90 seconds, then from your Mac: ssh pi@escort-bot.local # If hostname doesn't resolve: arp -a | grep -i "dc:a6\|b8:27\|d8:3a\|2c:cf"
First boot takes longer. If SSH fails: WiFi creds may be wrong — reflash.
Troubleshooting
# lgpio missing? sudo apt install python3-lgpio # tflite-runtime won't install? pip install --break-system-packages tflite-runtime # GPIO permission denied? sudo usermod -aG gpio pi && sudo reboot # Pi throttling under load? # Attach active cooler. Throttles at 85°C.
Bookworm uses lgpio (not RPi.GPIO) and enforces PEP 668 for pip.
Demo Day Performance Tips
# Overclock (needs active cooling): echo "arm_freq=2800" | sudo tee -a /boot/firmware/config.txt # Disable Bluetooth (saves CPU): sudo systemctl disable bluetooth # Pin CPU to max performance: echo performance | sudo tee \ /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor
Optional. Only apply after bot is stable and tested.
04 — Software (run in order)
Step 1 — Copy code to Pi
scp -r ~/hackathon/escort-bot/ pi@escort-bot.local:~/escort-bot/
Run from your Mac. Copies all code + scripts to Pi.
Step 2 — Install dependencies
ssh pi@escort-bot.local cd ~/escort-bot chmod +x install.sh ./install.sh
Installs picamera2, TFLite, gpiozero, lgpio. Downloads MobileNet SSD model. ~15 min.
Step 3 — Test camera
libcamera-hello --timeout 5000
Should run 5 sec without errors. If fail: check CSI cable seated firmly.
Step 4 — Test sonar
python3 -c "from gpiozero import DistanceSensor; \ s=DistanceSensor(echo=24,trigger=25); \ print(f'{s.distance*100:.1f}cm')"
Hold hand at 30cm → should read ~30cm. If always 0: check voltage divider wiring.
Step 5 — Test motors (wheels up!)
python3 -c "from gpiozero import Robot; import time; \ r=Robot(left=(17,27),right=(22,23)); \ r.forward(0.3); time.sleep(1); r.stop()"
Prop bot up first. All 4 wheels should spin forward. Swap wires if reversed.
Step 6 — Test pan/tilt
python3 pan_tilt.py --scan
Camera sweeps bottom to top. If servos jitter: check 5V power, try separate supply.
Step 7 — Run the bot
python3 main.py
Full escort mode. Stand in front → bot follows. Still 3 sec → triggers rack scan. Ctrl+C to stop.
Tuning (edit top of main.py)
# Oscillates left/right? KP = 0.6 # Doesn't turn enough? KP = 1.5 # Too close? TARGET_AREA_RATIO = 0.10 # Too far? TARGET_AREA_RATIO = 0.20 # Too slow? BASE_SPEED = 0.7
Tune on the floor. Save the values that work.
