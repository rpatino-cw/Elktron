System Architecture

Every subsystem connected through the Pi 5 GPIO and CSI bus. No external compute.

Compute
Raspberry Pi 5 (8 GB) Active Cooler MicroSD 32 GB
Vision
Arducam IMX708 (CSI) → YOLOv8n (Ultralytics) → Person bbox + centroid
Motion
L298N Dual H-Bridge 4x TT DC Motors (2 parallel pairs) Differential Tank Drive
Sensors
HC-SR04 Ultrasonic (obstacle avoidance)
Actuation
Pan-Tilt Servos (x2 MG90S) Pan GPIO 12 / Tilt GPIO 13
Interface
Pi 5 GPIO (40-pin header) → L298N IN1-IN4 → Motor Direction Control

Pi GND L298N GND (common ground)

Ultrasonic Sensor HC-SR04

Pi 5V (Pin 2) HC-SR04 VCC
GPIO 25 (Pin 22) HC-SR04 TRIG
GPIO 24 (Pin 18) HC-SR04 ECHO → 1kΩ divider
Pi GND (Pin 6) HC-SR04 GND + 2kΩ to divider
!
Voltage divider required on ECHO. The HC-SR04 outputs 5V on ECHO but Pi 5 GPIO is 3.3V tolerant. Use a 1kΩ + 2kΩ resistor divider to step down, or swap for an HC-SR04P (3.3V native).
