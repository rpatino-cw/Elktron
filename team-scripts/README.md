# Elktron — Team Scripts

Three scripts. That's it.

## First Time Setup

1. Download `setup.sh` (or Romeo will send it to you)
2. Open **Terminal** (Cmd + Space → type "Terminal" → Enter)
3. Drag `setup.sh` into the Terminal window and hit Enter
4. Done — the project is on your laptop

## Daily Workflow

**Get everyone's latest changes:**
```
cd Elktron
bash pull.sh
```

**Upload your changes:**
```
cd Elktron
bash push.sh
```

It will ask what you changed — type one sentence and hit Enter.

Tell Romeo. Don't try to fix git yourself.
